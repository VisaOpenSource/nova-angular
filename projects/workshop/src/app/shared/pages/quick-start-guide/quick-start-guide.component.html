Quick start guide

Quickly get started with a brand new Angular application.

Step 1: Update Angular

Nova Angular supports Angular 19, 20, and 21. See Angular's guide on how to upgrade Angular from your current version to the target version.

Step 3: Set up application

Import styles

Import the Nova Styles library and theme in your angular.json or equivalent file. If you would like a particular Nova theme, place it above the root in your project. The visa-light theme is the default. To see all the available themes see the themes docs inside our styles workshop.

Update compilerOptions

Add paths in your tsconfig.

Update your app.module

Add our Nova module to your app.module

Step 4: Add icons

Nova Angular can be used with Nova Icons or a custom icon library. This documentation site and the component examples uses the Nova Icons library.

Installing Nova Icons for Angular

Use standalone icons from @visa/nova-icons-angular (recommended)

Import and use the icons as you need them. See our Icon documentation for more examples.

    @for (item of iconImports; track item.id; let i = $index) {

Use icons sprite from @visa/nova-icons-angular

This will add all of the Nova Icons as a sprite into your project. You can then reference them as needed. You can use the Visa sprite, the Generic sprite, or both.

    @for (item of spriteImports; track item.id; let i = $index) {
