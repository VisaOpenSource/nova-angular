## [6.0.0](Learn more at: https://design.visa.com/developing/angular/changelog) (2025-08-07)


### ⚠ BREAKING CHANGES

* **switch-label:** merged with label
* **vTypographyColor:** migrated to prop in typography directive
* **opensInNewTab:** migrated directive to link as prop
* **disclosure-tab-item:** migrated directive to tab-item
* **button-disabled:** migrated directive to button
* **button-stacked-directed:** migrated directive to button
* **avatar-role-img:** removed directive
* **pagination-service:** removed service
* Combobox service's autoFilterBasedOnList is removed. Use autoFilter instead. PaginationService is also removed. Use PaginationControl instead.
TLDESDE-1148
* **tbody,tr:** removed directives
* **base-interactive-element:** removed directive
* **app-ready-service:** simplified service
* **visa-logo-constants:** removed LogoColor
* NovaLibService's routeChange observable and getCurrentRoute method are deprecated
* **panel:** converted to directive
* **dialog:** converted to directive, removed `cdkFocusTrap` package
* **base-interactive:** replaces output with EventEmitter
* AccordionButtonHeadingDirective with selector '[v-accordion-heading][v-button], [v-accordion-toggle-button]' is no longer in nova-angular
TLDESDE-1038
* toggle button component now only supports button children, radio and checkbox children will take on default radio and checkbox group behavior
TLDESDE-990
* **combobox:** must explicitly use v-floating-ui-container alongside v-combobox if relevant for your case
TLDESDE-95-
* **flex:** vGap no longer adds v-flex class by default, you will need to explicitly add a flex display directive if that is your intended css display
* **flex:** vGap no longer adds v-flex class by default, you will need to explicitly add a flex display directive if that is your intended css display
* remove built-in support for listbox directive with radio and checkbox component children TLDESDE-953
* private properties no longer use underscore
